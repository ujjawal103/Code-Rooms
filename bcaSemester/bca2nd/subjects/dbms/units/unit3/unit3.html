Intro to RDBMS

Learn about the Relational Database Management System (RDBMS), its structure, and how it manages data using relations (tables).

Integrity Constraints & Codd's Rules

This topic covers the rules that govern the validity and consistency of data in RDBMS, including:

  • Integrity Constraints – Entity Integrity, Referential Integrity, Domain Constraints, and Key Constraints
  • Codd’s Rules – The 12 rules proposed by E.F. Codd that define what qualifies a system as a true RDBMS

Relational Algebra & Calculus

Study the formal foundations of querying relational databases:

  • Relational Algebra – A procedural query language using operations like select, project, union, set difference, and join.
  • Relational Calculus – A non-procedural query language with two forms:
    • Tuple Relational Calculus (TRC)
    • Domain Relational Calculus (DRC)
